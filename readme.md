# SEzGP

This folder includes the experiments of SEzGP with the Examples 1 to 4.

---

### Examples
Ex1, Ex2, Ex3, and Ex4: Each scenario and data included.

### Models
Includes EzGP, EEzGP, and Vecchia approximation.

*   **EzGP**: Used for twin, NN, la, LE models.
*   **EEzGP**: Used for twin, NN, la, LE models using SEzGP.

> **Note**: The naming was initially HEzGP; we changed it to SEzGP in the final paper, so HEzGP refers to SEzGP.

*   **Vecchia**: Uses SVecchia by scaled Vecchia approximation (Katzfuss, Guinness, Lawrence) modified for HEzGP.

---

### Plots
Includes data collection, boxplots, and average time calculations.

### EzGP-Cpp
Includes Cpp implementations to reduce computation time, intended for integration into the EzGP package.

*   **Ex1**: Time comparison with data.
*   **fast_C_codes**: Modifications for the EzGP package.
*   **methods**: Original package and modified version codes for running examples.
